PCB Defect Detection using Machine Learning
 Overview

This project focuses on detecting defects in Printed Circuit Boards (PCBs) using Machine Learning and Image Processing techniques. The system classifies PCB images into defective and non-defective categories, helping automate quality inspection in manufacturing.

🚀 Features
📷 Image-based defect detection
🤖 Machine Learning / Deep Learning model
⚡ Fast and automated predictions
🌐 API integration for real-time detection
📊 Visualization of results
🛠️ Tech Stack
Python
NumPy, Pandas
OpenCV
Scikit-learn / TensorFlow / Keras
Flask / FastAPI (for backend)
HTML, CSS, JavaScript (for frontend)
